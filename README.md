# Token Reader for Exam

Aplikasi web sederhana untuk membacakan token ujian menggunakan Text-to-Speech API.

>  Bekerja dengan baik di Microsoft Edge

## Fitur
- Play & Pause kontrol
- Pengulangan otomatis 1-5x
- Atur kecepatan pembacaan
- Dark mode UI dengan desain modern

## Cara Menggunakan
1. Masukkan token ujian
2. Pilih suara Bahasa Indonesia
3. Atur kecepatan dan jumlah pengulangan
4. Klik Play untuk membacakan token

## Teknologi
- HTML5
- CSS3 (Dark Mode Design)
- Web Speech API
